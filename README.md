# VulnX

## Adavnced Web Vulnerability Scanner

VulnX is an automated web vulnerability scanner designed to identify common security weaknesses in web applications.

It combines multiple security assessment techniques into a single scanning engine to simplify reconnaissance and vulnerability detection.

---

## Features

- Automated Web Scanning
- Security Header Analysis
- Web Technology Detection
- Multi-tool Scanning
- Vulnerability Reporting
- Reconnaissance Automation
- 80 automated vulnerability checks
- Security Score
- Scan Metadata
- Executive Recommendations
- JSON Report Export
- TXT Report Export
- Windows Compatible
